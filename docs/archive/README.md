Retained **historical / handoff / working** documents whose task, spike, or
phase has concluded. They are kept for provenance but are **not living
reference docs** — do not treat them as current.

The living docs remain at the `docs/` top level: `architecture.md`,
`STATUS.md`, `CHANGELOG.md`, `future-plans.md`, `SECRETS.md`, and
`SOAK_RUNBOOK.md`.
